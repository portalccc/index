Nota metodológica

O indicador CUMPRIMENTO ANS permite o monitoramento mais amplo de uma série de dados conforme acima. Para ver dados relativos a cada unidade e o cumprimento do indicador, deve-se verificar a página de cada unidade. O indicador de cumprimento ANS é apurado a partir de processos concluídos, de modo automatizado. Não foram efetuados expurgos de processos com observações em etapas.
